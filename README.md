# Zhao Du Academic Homepage

This is a lightweight static personal homepage inspired by academic GitHub Pages sites.

## Files

- `index.html`: page content.
- `styles.css`: responsive layout and visual design.
- `assets/docs/Zhao_Du_CV.pdf`: CV linked from the profile and contact section.
- `assets/images/`: selected research figures used as publication thumbnails.

## Deploy to GitHub Pages

1. Create a GitHub repository named `YOUR_USERNAME.github.io`.
2. Upload the contents of this `personal-homepage` folder to that repository root.
3. In GitHub, open `Settings > Pages`, choose the `main` branch and root folder.
4. Visit `https://YOUR_USERNAME.github.io` after Pages finishes building.

Before deployment, replace any paper titles, links, and status labels with the final public versions you want displayed.
